== theories

This is based on https://jqwik.net/[jqwik]. This module provides sophisticated tests for algebraic structures, but also more straight forward things, like testing the contract of `Comparable`

Start with adding  link:https://central.sonatype.com/artifact/org.meeuw.math/mihxil-theories[`org.meeuw.math:mihxil-theories`] as a dependency. This would be the maven syntax):
[source, xml]
----
<dependency>
 <groupId>org.meeuw.math</groupId>
 <artifactId>mihxil-theories</artifactId>
 <scope>test</scope>
</dependency>
----
Your test implements the `-Theory` interface and provides the datapoints.

[source, java]
----
include::../mihxil-demo/src/test/java/org/meeuw/math/demo/QuickStartTheoryTest.java[]
----
That's all, you can run in your IDE or with `mvn test`
